# Foodie – Food Delivery App (Frontend Project)

Foodie is a modern and responsive front-end website designed using HTML5 and CSS3. It demonstrates practical web development skills including Grid and Flexbox layout systems, a clean user interface, and mobile-first responsive design principles.

This project aims to represent a food delivery service where users can browse food categories, explore restaurant options, view promotional offers, and read customer testimonials.



## Features

- Fully responsive layout (desktop, tablet, and mobile)
- Modern, clean, and visually consistent UI
- CSS Grid and Flexbox used for page layout
- Accessible and semantic HTML structure
- Promo offers section with coupon codes
- Testimonial cards including user avatars
- Dynamic food category and featured restaurant sections
- Hover animations and interactive UI behavior
- Social media icons included in the footer



## Technologies Used

- HTML5
- CSS3
- Google Fonts



## Responsiveness

Responsive breakpoints included:
@media (max-width: 900px)
@media (max-width: 600px)


All sections adjust according to screen size to maintain usability and readability on different devices.



## How to Run

1. Download or clone the repository.
2. Ensure all referenced images are located inside the project's image folder.
3. Open 'capstone.html' in any browser.
